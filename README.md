# 📊 HR Analytics Dashboard (Power BI)

## 📌 Overview
This project is an interactive **HR Analytics Dashboard** built using **Power BI** to analyze employee attrition and workforce trends. It provides actionable insights into employee behavior, helping organizations make data-driven HR decisions.

---

## 🚀 Key Features
- Employee count and attrition analysis
- Attrition rate calculation
- Department-wise insights (HR, R&D, Sales)
- Attrition breakdown by:
  - Age group
  - Education
  - Salary slab
  - Job role
  - Years at company
- Gender-based attrition comparison
- KPI cards for quick insights

---

## 📊 Dashboard Insights
According to the dashboard:

- 👥 Total Employees: **1417**
- 📉 Attrition Count: **226**
- 📊 Attrition Rate: **15.9%**
- 💰 Avg Salary: **6.5K**

- Highest attrition occurs in **Laboratory Technician & Sales roles**
- Majority attrition is in **26–35 age group**
- Employees earning **≤ 5K salary** show highest attrition
- **Life Sciences & Medical fields** contribute most to attrition
